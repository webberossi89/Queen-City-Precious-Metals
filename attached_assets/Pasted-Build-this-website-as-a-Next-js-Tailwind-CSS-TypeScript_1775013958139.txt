Build this website as a **Next.js + Tailwind CSS + TypeScript** project.

Use the existing brand colors already defined in the project. Do not create a new color palette.

Create a premium, responsive, SEO-friendly multi-page marketing website for **Queen City Precious Metals** with a polished, modern feel. The site should feel refined, trustworthy, local, high-value, and conversion-focused. Visually, it should feel closer to modern premium websites like Linear, Vercel, Notion, and Superhuman, adapted for a precious metals and luxury watch buyer.

Use:

* **Next.js App Router**
* **TypeScript**
* **Tailwind CSS**
* **Framer Motion** for subtle animations
* clean reusable components
* structured content/data files for events, FAQs, and buying categories

The website should include these pages:

* `/` Home
* `/what-we-buy`
* `/events`
* `/about`
* `/faq`
* `/contact`

Business details:
Queen City Precious Metals buys:

* gold jewelry
* silver jewelry / scrap precious metals
* coins and coin collections
* gold and silver bars / bullion
* estate jewelry
* luxury watches
* designer watches
* select Apple Watches

Important exclusions:

* do not mention antiques
* do not mention playing cards
* do not include those categories anywhere except one clarification on the homepage FAQ or qualification note saying they do not buy them

Priority items and brands to highlight visually and in copy:

* Rolex
* Cartier
* Breitling
* Apple Watches
* gold chains
* gold necklaces
* coins
* bars / bullion

Main website goals:

1. clearly show what the company buys
2. build trust and professionalism
3. prominently feature upcoming events
4. drive users to either attend an event or contact the business about an item

Primary CTAs:

* View Upcoming Events
* See What We Buy
* Contact Us
* Ask About Your Item

Use a strong reusable site structure:

* sticky header

* reusable hero sections
* reusable CTA sections

* reusable section wrapper / content container components

Create structured data files such as:

Use tasteful placeholder images or gradient/image panels where needed. If actual photography is not available, build elegant visual blocks that suggest:

## Global design direction

The design should be:

* premium
* modern
* elegant
* local
* trustworthy
* clean
* conversion-oriented

Use the existing brand colors from the project. Build the styling system around them with:

* strong typography hierarchy
* generous spacing
* rounded cards
* subtle borders
* soft shadows
* premium hover states
* accessible contrast
* mobile-first responsiveness

Include tasteful visual motifs such as:

* subtle Charlotte skyline line art
* watch-inspired circular accents
* metallic/gold glow highlights
* soft grid textures
* coin-edge style circular detailing
* blurred gradient shapes behind key content

Do not make the site feel like a pawn shop, flea market, or cluttered “roadshow” site.

## Motion direction

Use Framer Motion for subtle animations only:

* fade-up reveals
* staggered card reveals
* slight hover lift
* soft CTA hover transitions
* restrained shimmer or glow
* very minimal parallax only if tasteful

Avoid:

* bounce animations
* flashy transitions
* loud effects
* aggressive parallax
* anything gimmicky
